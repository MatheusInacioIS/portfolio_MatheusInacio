# Batalha de modelos e engenharia de prompt
 
## 📝 Descrição do Projeto
Esta atividade consiste em uma análise comparativa de diferentes Large Language Models (LLMs), focada em mensurar a precisão técnica e a capacidade de resposta a diretrizes rigorosas.
O objetivo principal é verificar a eficácia dos modelos em operar sob um conjunto estrito de instruções, garantindo que a saída gerada não apenas cumpra o conteúdo solicitado, mas respeite integralmente as restrições formativas e lógicas impostas.

## 🚀 Tecnologias Utilizadas
* **Linguagem:** Diferentes (LLMs) como Claude, Gemini, Chatgpt, Grok entre outras.
 
## 📊 Resultados e Aprendizados
A atividade proporcionou uma análise comparativa profunda, demonstrando disparidades significativas na capacidade de resposta e no controle operacional de diferentes modelos.
* **Taxa de Conformidade:** O modelo de maior desempenho atingiu 95% de aderência estrita às instruções condicionais, processando corretamente restrições complexas sem desvios de protocolo.
* **Precisao tecnica** Implementei uma matriz de avaliação para medir a fidelidade das respostas, garantindo que o refinamento linguístico não resultasse em perda de precisão factual ou lógica.
* **Mapeamento de Instruções:** Desenvolvi testes de estresse que analisam a sensibilidade dos modelos a comandos conflitantes, identificando quais arquiteturas conseguem priorizar regras de segurança e formatação sob alta demanda de processamento.
 
## 🔧 Como Executar
1. Criação do Prompt: Estruture um comando contendo o contexto do projeto e o conjunto estrito de instruções técnicas para o teste.
2. Definição das LLMs: Escolha os modelos de linguagem que serão submetidos à avaliação (ex: Gemini, GPT-4, Claude).
3. Execução do Teste: Insira o prompt padronizado em cada modelo selecionado, mantendo a uniformidade para garantir a validade do benchmark.
4. Registro de Resultados: Documente as respostas obtidas, comparando o nível de precisão e conformidade de cada modelo em relação às regras estabelecidas.
