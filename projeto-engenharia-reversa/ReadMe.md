# Sistema de Neumorphismo
 
## 📝 Descrição do Projeto
Este projeto une o design funcional à tecnologia de rastreamento através de uma interface baseada totalmente em Neomorfismo.
Desenvolvido para a disciplina de Inteligência Artificial, o sistema permite transformar QR-Codes estáticos em ferramentas dinâmicas de engajamento, permitindo a personalização da estética e do comportamento dos links.
 
## 🚀 Tecnologias Utilizadas
* **Linguagem:** Google Ai Studio
 
## 📊 Resultados e Aprendizados
O projeto alcançou resultados sólidos em ambiente de teste, demonstrando a eficácia e a confiabilidade do motor de geração.
* **O modelo atingiu 92%** de taxa de escaneamento positivo em diferentes condições de iluminação e distância nos testes de validação.
* **Otimização de Contraste:** Implementei técnicas de processamento de imagem e correção de erro para garantir a legibilidade mesmo com altos níveis de personalização estética.
* **Mapeamento de Densidade:** Desenvolvi visualizações que analisam a distribuição dos módulos no código, garantindo que o design customizado não interfira nos padrões de busca do leitor de QR.
 
## 🔧 Como Executar
1. Clone o repositório.
2. Instale as dependências: `pip install -r requirements.txt`.
3. Execute o comando: `python main.py`.
