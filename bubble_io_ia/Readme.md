# Sistema de Gerenciamento de Solicitações de Reembolso
 
## 📝 Descrição do Projeto
ESistema web desenvolvido utilizando a plataforma no-code Bubble.io para gerenciamento de solicitações de reembolso.
O sistema permite que usuários criem, acompanhem e organizem pedidos de reembolso de forma simples e eficiente, com autenticação, controle de privacidade e fluxos automatizados.

## 🎯 Objetivo
Desenvolver uma aplicação funcional utilizando Bubble.io, aplicando conceitos de Engenharia de Software, modelagem de dados, workflows automatizados e boas práticas de organização de sistemas web sem programação tradicional.

## 🚀 Tecnologias Utilizadas
* **Linguagem:** Qwen3.5-Plus
* **Estilos Visuais:** Barroco, Hiperrealismo e 4K
 
## 📊 Resultados e Aprendizados
O projeto demonstrou como o detalhamento de instruções transforma resultados genéricos em produções de alta fidelidade técnica e emocional.
* **Refinamento de Imagem:** O uso de prompts específicos permitiu a criação de um astronauta com trajes barrocos tocando violoncelo em Marte, evoluindo de uma descrição simples para uma configuração hiperrealista em 4K.
* **Narrativa Adaptativa:** Foram desenvolvidas cinco variações de e-mails formais de piratas para reis, cada uma aumentando a complexidade do contexto (desde pedidos simples até súplicas urgentes para evitar a execução).
* **Estruturação de Dados:** Implementação de cabeçalhos formais detalhados, incluindo prioridades de entrega, coordenadas geográficas (como $50^{\circ}N$ 1°W) e prazos rigorosos de 48 horas.
 
## 🔧 Como Executar
1. Selecione um dos modelos de prompt listados no arquivo (ex: Prompt 1 ao 5).
2. Insira o comando no motor de IA desejado (Qwen3.5-Plus recomendado).
3. Para resultados visuais, utilize as instruções de estilo artístico e resolução 4K especificadas.
